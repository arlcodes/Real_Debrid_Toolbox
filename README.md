# Real_Debrid_Toolbox

Python Script to automate Real Debrid actions via API

Main functions:
1 - Upload torrent files (converts to magnets) in current dir

2 - Check status of in progress torrents

3 - Download completed torrents

4 - Download and unrestict links from hosters

5 - De Dupe torrents via hash check

6 - Renew sub (via referal url http://real-debrid.com/?id=3488563)


Required packages (non defaults)
requests (pip install requests)
tqdm (pip install tqdm)

Run via Start RD Toolbox batch file

Tested on Python 3.13.3, W11 Pro
